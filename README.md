# KopKopMusic

Local Müzik Çalar Uygulaması

## Özellikler

- Local mp3 Dosyalarını çalabilir
- Metadata verilerini gösterir
- Multimedia işlemlerini yürütebilir
- Tekrar ve karışık çalma özelliği
- Karanlık ve aydınlık tema desteği
